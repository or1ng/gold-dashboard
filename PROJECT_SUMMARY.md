## 一、项目概述

### 1.1 项目目标

构建一个**黄金价格实时监控与交易决策辅助看板**，具备以下核心能力：

| 目标 | 说明 |
|------|------|
| 📊 实时数据展示 | 展示黄金价格、美元指数、美债收益率等关键指标 |
| 🎯 智能信号判定 | 基于阈值自动判断多空方向（红=利空/绿=利多/黄=震荡） |
| 📰 财经资讯聚合 | 接入金十数据快讯和财经日历 |
| 💡 买卖时机建议 | 根据综合指标给出操作建议 |
| 🖥️ 可视化看板 | 玻璃态UI设计，支持响应式布局 |

### 1.2 需求来源

用户提供了一套完整的黄金投资分析框架，包含：

- **核心定价锚**：美元指数 DXY、10年期美债实际收益率 TIPS
- **货币政策风向标**：美联储议息会议、CPI/非农数据
- **短期情绪催化剂**：地缘危机、原油价格
- **资金底牌**：COMEX 黄金非商业净持仓、ETF 持仓
- **极简速判口诀**：双绿上车 / 双红跑路 / 地缘不追

#### 4.3 ⚠️ 关键问题：DXY 数据系列的修正

**发现的问题**：

最初使用的是 **DTWEXBGS**（广义贸易加权美元指数），该指数：
- 基准：2006年1月 = 100
- 正常波动区间：**95 ~ 130**
- 用户设定的阈值 100/105 **不适用于此指数**

**修正方案**：

改用 **DTWEXAFEGS**（先进经济体贸易加权美元指数），该指数：
- 只包含发达经济体货币（欧元、日元、英镑、加元、瑞典克朗、瑞士法郎）
- 与市场标准的 **DXY 期货指数高度相关**（相关系数 > 0.95）
- 正常波动区间：**85 ~ 115**
- 用户设定的阈值 **100/105 完全适用**

| 对比项 | DTWEXBGS（旧·错误） | DTWEXAFEGS（新·正确） | 市场标准DXY |
|--------|-------------------|--------------------|------------|
| 货币篮子 | 包含新兴市场 | 仅发达经济体 | 6种主要货币 |
| 波动区间 | 95-130 | 85-115 | 80-120 |
| 阈值100/105适用性 | ❌ 不适用 | ✅ 适用 | ✅ 适用 |
| 与黄金相关性 | 中等 | 高 | 高 |

**此修正是项目中的一个重要技术决策。**

#### 4.4 信号判定逻辑

```python
# DXY 信号判定
if dxy_value <= 100:
    signal = "bullish"      # 🟢 强利多（做多）
elif dxy_value >= 105:
    signal = "bearish"      # 🔴 强利空（做空）
else:
    signal = "neutral"      # 🟡 震荡观望

# TIPS 信号判定
if tips_value <= 0.5:
    signal = "bullish"      # 🟢 多头占优
elif tips_value >= 1.5:
    signal = "bearish"      # 🔴 空头占优
else:
    signal = "neutral"      # 🟡 中性偏空

# 综合评分算法
score = 0
if dxy == bullish: score += 50
if tips == bullish: score += 50
# 最终评分 0-100
```

### 常见问题

| 问题 | 解决方法 |
|------|---------|
| 显示"数据加载失败" | 确认 Python 服务正在运行 |
| 端口 8001 被占用 | 修改 data_service_jin10.py 中的端口号 |
| 金十数据获取失败 | 检查网络连接，确认 API Token 未过期 |
| FRED 数据为空 | 确认 API Key 有效，检查是否超出频率限制 |
